    An open-source and extensible, stand-alone presentation app for developers

    Currently in alpha. Built in Electron with React-Redux/TypeScript.

    Download for Mac

    Code highlighting

    No more switching windows in the middle of your presentation. Seamlessly highlight code as you focus on communicating what your code does.

    Extensible

    Add new functionality by creating your own plugins. Access the component API and harness the power of NodeJS to bring your dream feature to life.

    Upcoming features

    We're actively working on devdecks. Some of the features we envision in future releases include the ability to make API calls, integrate voice recognition and bluetooth device connectivity, collaborate remotely, and automatically sync files with cloud storage services.
